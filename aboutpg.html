About The Site

This website was created from scratch in the free, open-source Brackets HTML editor. Brackets is developed in HTML, CSS and JavaScript, making anyone who uses the editor also capable of tweaking the editor by writing extensions.

We are using GitHub Pages to host the site. GitHub Pages was chosen because it is free, fast, reliable and simple.
